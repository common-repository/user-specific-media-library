Limit users' media library to media they have uploaded.

== Description ==

Limit users' media library to media they have uploaded. Users with the edit_others_posts capability (generally editors and administrators) have access to all media.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
